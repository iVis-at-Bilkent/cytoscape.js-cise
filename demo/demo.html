CiSE Demo Help

With this demo, you can:
  • Import a graph in GraphML format. If you'd like your graph to be automatically clustered on load, check "Apply Markov Clustering on Load"
  • Lay it out using CiSE style
  • Right-click on a node to add some random neighbors
  • Change current clustering using Add/Remove Cluster buttons
  • Adjust layout options (please refer to the API in README for an explanation of these options)
